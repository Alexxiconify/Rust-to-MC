# Rules

1. **Code**: Bug-proof. Clean comments.
2. **Docs**: Pro MD. Normal writing.
3. **Output**: Use [`docs/output-style-guide.md`](docs/output-style-guide.md).
4. **Env**: Win11/PS7. No linux: grep, rm, sed.
5. **Tool**: API > Shell.
6. **Map**: Update `ROADMAP.md` post-fix.
7. **Action**: Move fast. Act now.
8. **Lint**: After every `.rs`/`.java` edit, fix all Clippy warnings and Java IDE warnings. Zero warnings on exit.
